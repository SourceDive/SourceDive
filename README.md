## 👋 关于我们

**SourceDive** 是专注于开源项目与代码阅读的技术组织。我们致力于分享、分析和探索优质开源项目，帮助开发者提升代码阅读能力，拥抱技术成长。

## 🚀 代表性项目

- 仓库列表请见 [SourceDive · Repositories](https://github.com/orgs/SourceDive/repositories)
- 我们推荐你探索每个项目的 README 和 Issues 区，参与讨论和学习。
